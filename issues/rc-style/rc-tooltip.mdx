## Install `rc-tooltip` component

<Tabs>

```bash
npm install rc-tooltip
```

  </TabItem>
  <TabItem value='yarn' label='yarn'>

```bash
yarn add rc-tooltip
```

  </TabItem>
</Tabs>

## Add CSS CDN into html file

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/cdnhust/rcstyles@master/rc-tooltip.css" />
```
